## Rules
A timer will start after the player attempts to match the first card flipped over.
A star rating for the player's performance is graded by 1 star, 2 stars, or 3 stars.
The player must match all the cards in under sixteen moves to recieve 3 stars, under 21 moves to recieve 2 stars, and will receive 1 star if the game is completed in over 20 moves.
 

## Instructions

Players can use the refresh button next to the timer to reshuffle the cards, reset the timer, and reset the move counter.
